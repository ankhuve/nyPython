Den här katalogen innehåller en P-uppgift som behöver granskas.

OBS: Uppgiften ska endast om man inte har något annat program att granska.

Programmet fungerar men har flera brister. Man måste till exempel
titta på programkoden för att kunna räkna ut hur man startar
programmet. Gå igenom programmet noggrannt och provkör det. Fyll
sedan i ett granskningsprotokoll som du tar med på redovisning av granskningen.

För att starta programmet skriv "python FiktivPupg.py".
